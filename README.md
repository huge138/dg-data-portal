# dg-data-portal

数据门户系统 - 基于 Vue 3 + Vite + Element Plus + TypeScript 构建

## 项目结构

```
dg-data-portal/
├── src/
│   ├── components/          # 组件目录
│   │   └── SimpleDialog/    # 简单对话框组件
│   ├── router/              # 路由配置
│   ├── styles/              # 全局样式
│   ├── views/               # 页面视图
│   │   └── home/            # 首页
│   ├── App.vue              # 根组件
│   ├── main.ts              # 入口文件
│   └── vite-env.d.ts        # 类型声明
├── index.html               # HTML 模板
├── package.json             # 项目配置
├── tsconfig.json            # TypeScript 配置
├── vite.config.ts           # Vite 配置
└── .env                     # 环境变量
```

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

### 预览生产构建

```bash
pnpm preview
```

## 功能特性

- ✅ Vue 3 + Composition API
- ✅ TypeScript 支持
- ✅ Vite 构建工具
- ✅ Element Plus UI 组件库
- ✅ Vue Router 路由管理
- ✅ Pinia 状态管理
- ✅ Sass 预处理器

## 技术栈

- Vue 3.5.34
- TypeScript 6.0.3
- Vite 8.0.10
- Element Plus 2.13.7
- Vue Router 5.0.6
- Pinia 3.0.4
